## Getting Started

First, install dependencies:

```bash
npm install
# or
pnpm install
# or
yarn install
# or
bun install
```

Second, run the development server:

```bash
npm run dev
# or
pnpm dev
# or
yarn dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
